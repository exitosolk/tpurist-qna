# Open Graph Image Specifications

Create the following images for social media sharing:

## 1. Default OG Image (`og-image.png`)
- **Size**: 1200 x 630 pixels
- **Format**: PNG or JPEG
- **Content**: 
  - OneCeylon logo/branding
  - Tagline: "Sri Lanka Travel Q&A Community"
  - Beautiful Sri Lanka imagery (beach, tea plantations, or temples)
  - Brand color: #2563eb (blue)

## 2. Twitter Card Image (`twitter-image.png`)
- **Size**: 1200 x 630 pixels (same as OG)
- **Format**: PNG or JPEG
- **Content**: Same as OG image

## Design Tips:
- Keep important content in the center 1000x506px area (safe zone)
- Use high-quality images of Sri Lankan landmarks
- Ensure text is readable at small sizes
- Test on Facebook, Twitter, and LinkedIn preview tools

## Tools to Create Images:
- Canva (canva.com) - Easy templates
- Figma (figma.com) - Professional design
- Adobe Photoshop
- Online tools: og-image.xyz, cloudinary.com
